# Xcode Preivew 錯誤

使用 Xcode preview SwitUI 失敗, 跑出以下錯誤時解決方法

> 蔬店 does not contain a preview provider named "MapContainer_Previews". Check your build settings to ensure the preview provider is compiled into your product

**1. 檢查 Target Building Setting `Product Name` 是否為中文**

如果為中文, 就改成非中文.  
建議改成 `${TARGET_NAME}`

**2. 設置 App 名稱**

由於 Step1 改變了 `Product Name`, 所以 App 名稱跟著改變了.  
在 info.plist 使用 `Bundle display name` 來設置 App 名稱.
