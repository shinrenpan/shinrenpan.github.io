簡單實作 [SMART on FHIR](https://docs.smarthealthit.org/) OAuth, 並取得病人資料.

[Demo](https://github.com/shinrenpan/FhirDemo)

![](/images/2025-03-18/01.gif)

## 步驟

1. 透過 ASWebAuthenticationSession 取得 code.
    
2. 透過 code 換取 token.
    
3. 使用 API 取得病人資料.
    
4. 透過 [FHIRModels](https://github.com/apple/FHIRModels) 解析.
    

### SMART Launcher 設置

1. 選一個 Standalone Launch.
    
2. Copy URL, 去掉 /fhir 當作 Base URL.
